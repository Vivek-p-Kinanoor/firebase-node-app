# **App Name**: Bhasha Guard

## Core Features:

- Malayalam Spell Check: Spelling and grammar check for Malayalam text using generative AI tool.
- English Hashtag Spell Check: Spelling check for English hashtags and keywords using generative AI tool.
- Name Check: Name check to verify if the given name follows conventional naming standards using generative AI tool.
- Error Display: Display identified spelling errors and suggestions for correction.
- Input Fields: Input fields for Malayalam text, English hashtags/keywords, and name.

## Style Guidelines:

- Primary color: Light purple (#BFA0E0), for a sense of calmness, precision, and carefulness.
- Background color: Light grey (#F0F0F2), almost white but softer and easier on the eyes for reading long blocks of text.
- Accent color: Light blue (#A0BFE0) to bring contrast, while still suggesting clarity and attention to detail.
- Body and headline font: 'Literata' (serif), lending a feel of careful, literary attention.
